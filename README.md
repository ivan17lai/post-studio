# igapp
自製的 IG 貼文編輯 App，以簡潔操作為核心設計理念。
支援多頁貼文排版、自訂背景色彩，以及針對 IG 優化的匯出選項。
目前正在開發 Ultra HDR 編輯與輸出支援。

## 主要功能

- 多種頁面比例選擇（4:5、3:4、4:3、5:4）
- 多頁貼文跨頁編輯
- 背景色彩自訂
- 匯出解析度選擇（IG 標準 1080 / 高畫質 2.4K）
- 從最後一頁反向輸出，方便發文時的照片選擇
